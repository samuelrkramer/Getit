# GETit

GETit is a social media page for sharing and viewing jokes, and engaging with them with comments. Both POSTs and comments are fully CRUD-capable. Technologies used for the backend include Python, Flask, PostgreSQL, SQLAlchemy, and Alembic. For the frontend, the technologies include JavaScript, Node.js, React, Redux. Docker is used to containerize the whole app for ease of deployment. Please visit at https://skgetit.herokuapp.com/

It's Reddit but for jokes. **GET it?**
